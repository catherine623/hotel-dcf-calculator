## 使用方法

1. 用手机扫描二维码，或直接在浏览器打开上面的链接
2. 页面加载后，可选择预设案例（华住REIT/锦江REIT/雅致酒店）
3. 也可手动输入参数
4. 点击「立即计算估值结果」查看完整DCF估值报告

## 功能说明

- ✅ 基于华住&锦江REIT招募说明书方法论
- ✅ 两个已验算案例（华住上海3.56亿/锦江广州1.96亿）
- ✅ 支持自定义参数
- ✅ 逐年现金流明细 + 图表可视化
- ✅ 敏感性分析
- ✅ 收益资本化法双重校验
